Sample Application

The example application is packaged as a WAR file and can be downloaded here. Note: Ensure that your browser does not change the file extension or append an additional one.

To run this application, simply move the WAR file to your CATALINA_BASE/webapps directory. A default Tomcat installation will automatically unpack and deploy the application. You can access it at the following URL (assuming Tomcat is running on the default port 8080):

http://localhost:8080/sample

If you want to explore the contents of the WAR file, you can extract it using the following jar command: 

jar -xvf sample.war

Note: CATALINA_BASE typically refers to the directory where you unpacked your Tomcat distribution. For more details on CATALINA_HOME, CATALINA_BASE, and their differences, refer to the RUNNING.txt file in your Tomcat distribution folder.
